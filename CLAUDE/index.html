Features What Makes Me Special

Constitutional AI

Anthropic의 Constitutional AI 방식으로 학습되어, 안전하고 윤리적인 응답을 생성합니다. 해로운 콘텐츠를 거부하면서도 최대한 도움이 됩니다.

Long Context Window

최대 200K 토큰의 컨텍스트 윈도우로, 긴 문서 전체를 한 번에 분석하고 이해할 수 있습니다. 책 한 권도 한 번에 읽습니다.

Reasoning & Analysis

단계별 논리적 사고를 통해 복잡한 문제를 풀어냅니다. 수학, 코딩, 논리 문제에서 뛰어난 추론 능력을 보여줍니다.

Honest & Transparent

모르는 것은 모른다고 솔직히 말하고, 확실하지 않은 것은 불확실성을 표현합니다. 거짓 정보를 만들어내지 않으려 노력합니다.

Tool Use & Agentic

코드 실행, 파일 읽기/쓰기, 웹 검색, API 호출 등 다양한 도구를 활용하여 실제 작업을 자율적으로 수행합니다.

Vision (Multimodal)

이미지, 차트, 다이어그램, 스크린샷을 이해하고 분석합니다. 시각 정보와 텍스트를 결합한 멀티모달 추론이 가능합니다.
